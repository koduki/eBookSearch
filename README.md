0. Install [Mechanize for Scala](https://github.com/koduki/mechanize)

0. Install [MongoDB](http://www.mongodb.org/)

1. Launch [SBT](http://code.google.com/p/simple-build-tool).

        sbt

2. Run Jetty

        container:start
        container:stop 
        ~compile
3. Go to http://localhost:8080/.


5. Happy hacking!
